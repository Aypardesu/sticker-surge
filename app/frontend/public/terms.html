Privacy

Cookies are used on this site in order to provide basic authentication functionality and for Google Analytics.

All user/server pages on this site are public and do not require authentication to view. This is necessary in order to appease users who do not wish to log in to the site but still need to view which stickers are available to them.

The following data is stored:

  • Your server name, server icon, and server ID is stored when the bot is added to the server and updated in real time while the bot is present. This is so that a page can be generated for your server (Viewable whenever someone uses the $stickers command in Discord or logs into the site and clicks on "Servers".)
  • The name and avatar of a user is saved if (and only if) a user private message the bot or logs into the website. This is so that a page can be generated for the user, which is viewable whenever they private message the bot with the $stickers command in Discord or log into the site and click "Personal Stickers".
  • IDs of users with special permissions (Server Managers, Sticker Managers, Whitelisted/Blacklisted Members) in your server are saved so that the bot knows who is able to perform what actions (Add stickers, Delete stickers, Manage sticker roles etc.)

This service does not collect any personal or private information. All aforementioned information is stored by the service for the explicit purpose of providing functionality.
